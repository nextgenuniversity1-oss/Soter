## Build Artifact

The WASM artifact was built with Stellar CLI 26 using the `wasm32v1-none` target (the CLI manages the build internally via `stellar contract build`):

```bash
stellar contract build
# Output: target/wasm32v1-none/release/aid_escrow.wasm
```

Or equivalently with cargo directly:

```bash
cargo build --release --target wasm32v1-none -p aid_escrow
# Output: target/wasm32v1-none/release/aid_escrow.wasm
```
